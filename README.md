# Інструкція з використання програми

 **Запуск програми:**

   - Використайте термінал:
     ```bash
     javac StringProcessor.java
     java StringProcessor
     ```

## Результат виконання
Програма виведе:
- Початковий текст
- Параметри пошуку підрядків
- Оброблений текст з видаленими підрядками
